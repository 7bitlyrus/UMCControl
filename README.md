# UMCControl
Interface for controlling Pterodactyl and your minecraft server from Discord.

## Quick Setup Guide
```sh
$ git clone https://github.com/IanMurray/UMCControl.git
$ cd UMCControl
$ npm install
$ cp example_config.json config.json
$ # Edit config.json to your liking
$ node index.js
```
